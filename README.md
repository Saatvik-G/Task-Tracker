# TaskSphere 🪐

[![GitHub license](https://img.shields.io/github/license/Saatvik-G/Task-Tracker)](https://github.com/Saatvik-G/Task-Tracker/blob/main/LICENSE)
[![GitHub Pages](https://img.shields.io/badge/deployed-GitHub%20Pages-blueviolet)](https://github.com/Saatvik-G/Task-Tracker)

A premium, responsive, and aesthetically outstanding personal Kanban task management board designed to keep you productive and focused. Built with modern dark-mode glassmorphism aesthetics and smooth interactive micro-animations.

---

## 🎨 Visual Experience & Theme

TaskSphere is designed to replace generic, boring task sheets with an interactive experience:
- **Intelligent Splash Screen**: A beautiful intro deck that welcomes you and transitions smoothly on clicking the button or pressing `Enter`.
- **Glassmorphism Design System**: Sleek frosted cards, subtle gradients (from Electric Indigo to Royal Violet), and responsive side panels.
- **Dynamic Stats & Progress Ring**: Watch your workflow progression in real-time with an SVG circular indicator showing task completion percentages.

---

## ✨ Features

- **Kanban-Style Columns**: Organize tasks under **To Do**, **In Progress**, and **Completed** statuses.
- **Full CRUD Management**: Seamlessly add, edit details, rename, or delete tasks.
- **Filtering & Search**: Live-filter tasks by tag categories (Work, Personal, Shopping, Health) or search titles/descriptions in real-time.
- **Local Persistence**: All tasks are stored securely inside your browser's local storage (`localStorage`) so your data remains intact across sessions.
- **Zero Configuration Hosting**: Completely static application designed to run out-of-the-box on services like **GitHub Pages** (no Node/localhost:3000 servers required).

---

## 🚀 How to Run Locally

Since TaskSphere is built using vanilla HTML, CSS, and JS, you don't need any complex installation.

1. Clone this repository:
   ```bash
   git clone https://github.com/Saatvik-G/Task-Tracker.git
   ```
2. Open `index.html` in your web browser of choice.

---

## 🌐 Deploy to GitHub Pages

To make it accessible online:
1. Go to your repository **Settings** on GitHub.
2. Navigate to the **Pages** menu on the left.
3. Under **Build and deployment**, select **Deploy from a branch** as the source.
4. Select the **`main`** branch and `/ (root)` folder, then click **Save**.
5. Your app will be live at `https://saatvik-g.github.io/Task-Tracker/`!
